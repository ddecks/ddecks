# Hey 👋

I build CLI tools, tinker with hardware projects, and study for AWS certs in my terminal. I like solving problems with minimal, practical tools — if it needs a build system, it's probably too complex.

## What I'm working on

🎯 **[cert-drill](https://github.com/ddecks/cert-drill)** — A Rust CLI for certification exam prep that captures your *reasoning*, not just your answers. Exports structured data for AI-assisted review. Works on Linux, macOS, Windows, and Termux (Android).

📚 **[bookwiki](https://github.com/ddecks/bookwiki)** — Feed it a novel, get a character wiki. Built it to keep track of characters across a 12-million-word web serial. Uses NLP for dialogue attribution, relationship graphs, and chapter-by-chapter tracking.

🔧 **[scrappy-solutions](https://github.com/ddecks/scrappy-solutions)** — A grab bag of small CLI tools and scripts for common engineering tasks. Nothing fancy — just things that save time.
